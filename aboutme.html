June 21st, 2017

The "Obamicon" project was fun to work on. It was great to be able to make a photo more interesting with just a few lines of code.

Most of the challenges that came with this project were because small syntax errors broke the whole program.

Once again, although the command prompt will identify the first "problem" area, it is difficult to understand how to fix the error. Trial and error was the easiest solution.

Each of the "Core 4" (loops, variables, functions, conditionals) were implemented into the project, except for a function. A loop was used to cycle through every pixel of a specified image. A conditional was used to assign a pixel a new color, based on its overall brightness. A variable was used to assign every pixel a (new) RGB value.

If we had more time, I would have tried to create a filter that changes the overall color of the picture. They look cool :D

The filters were probably the highlight of the day.
